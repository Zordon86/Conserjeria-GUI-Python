Una aplicación de escritorio desarrollada en Python para gestionar de forma eficiente la recepción y entrega de paquetes en una comunidad de vecinos.

## 🚀 Características Principales

* **Registro de Paquetes:** Interfaz gráfica para dar de alta nuevos paquetes indicando piso, empresa repartidora y nombre del vecino.
* **Control de Inventario:** Visualización en tiempo real de los paquetes pendientes de entregar mediante Pop-ups.
* **Gestión de Entregas:** Sistema de búsqueda y eliminación de paquetes una vez recogidos por el vecino.

* **Historial Persistente:** Registro automático de todas las entregas realizadas.
* **Almacenamiento Local:** Los datos se guardan en formato `.json`, asegurando que la información no se pierda al cerrar el programa.

## 🛠️ Tecnologías y Conceptos Aplicados

* **Lenguaje:** Python 3
* **Interfaz Gráfica:** Tkinter
* **Persistencia de Datos:** Módulo `json`
* **Arquitectura:** Programación Orientada a Objetos (POO) y principios de Clean Code (DRY).

## ⚙️ Cómo usar este proyecto

1. Descarga los archivos `.py` en una misma carpeta.
2. Ejecuta el archivo principal (el que contiene la interfaz).
3. El programa creará automáticamente los archivos `.json` necesarios en tu ordenador al realizar el primer registro.
